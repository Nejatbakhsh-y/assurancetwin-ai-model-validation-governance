# AssuranceTwin AI: A Reproducible Model Validation and AI Governance Framework for High-Stakes Financial AI

Independent model validation and AI governance framework for high-stakes financial AI using public HMDA mortgage data.

## Project Purpose

This repository develops a reproducible model validation and AI governance framework for mortgage lending risk analytics.

The project combines independent model validation, model-risk-management controls, fairness testing, explainable AI, drift monitoring, stress testing, challenger-model analysis, and governance documentation.

## Core Themes

- Independent model validation
- AI governance
- Model risk management
- Fair lending analytics
- Bias testing
- Explainable AI
- Drift monitoring
- Financial AI
- HMDA mortgage data
- High-stakes AI systems

## Repository Structure

| Folder | Purpose |

| configs/ | Project configuration files |

| notebooks/ | Optional exploratory notebooks |
| reports/ | Figures, tables, and final reports |
